# Dine on Campus
Food blog for close by restaurants.
<h1>Description</h1>
Hello, for the second assignment I am creating a small food blog that will highlight establishments close to campus. It will include a short description of each restaurant, some pictures of the dishes they offer and where exactly you can find them!
<h1>Dependencies</h1>
